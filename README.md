# TrustSplitz
A simple way to split trip expenses and settle via UPI.

## Project Setup

The frontend is built with React and Vite. Follow these instructions to set it up:

1. Clone repository
```bash
git clone <repo-url>
```

2. Install dependencies
```bash
npm install
```

3. Run development server
```bash
npm run dev
```

4. Open browser
Navigate to `http://localhost:5173` in your browser.
